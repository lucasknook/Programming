was correct.

# Tetris (WIP)
Tetris using the Super Rotation System.

Controls:  
A and D: move piece left or right.  
S: soft drop.  
W: hard drop.  
ENTER: rotate piece clockwise.  
RIGHT SHIFT: rotate piece counter clockwise.  
C: hold piece.
